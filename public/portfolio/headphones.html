Headphones UI Layout Project

View Project View Code

This is a UI layout design of a website that I created from a .PSD file. The .PSD file was only a desktop mockup, so I started the design with a mobile-first approach. I also used a component type approach when designing the site to help efficiently reuse components in the design with CSS 3. I implemented newer CSS 3 properties like flex-box module layout and the grid property as well.
