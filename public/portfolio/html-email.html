HTML Email Design Project

View Project View Code

This is the design of an HTML email that I created from a .PSD file. The .PSD file was only a 700 pixels design mockup, so I started the design with a mobile-first approach. With this design, no newer CSS 3 properties like flex-box module layout or grid properties are used. The design only uses tables to make the layout mobile responsive and all styles are done inline. All properties that were used to make the HTML email design were verified for full compatibility with emails.
